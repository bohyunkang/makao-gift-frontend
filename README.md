## 👩🏻‍💻 프로그램 실행하는 법
```
1️⃣ 레포지토리 클론
git clone https://github.com/bohyunkang/makao-gift-frontend.git

2️⃣ 디렉터리 이동
cd makao-gift-frontedn

3️⃣ 패키지 설치
npm install

4️⃣ 프로그램 실행
npm start

5️⃣ 유닛 테스트 확인하는 방법
npm test

6️⃣ 인수 테스트 확인하는 방법
npm run codeceptjs:headless // 터미널에서 확인 가능
npm run codeceptjs:ui // UI 화면으로 확인 가능
```
